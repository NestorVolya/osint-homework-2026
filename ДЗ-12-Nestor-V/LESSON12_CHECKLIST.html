Розділ 4 — Neo4j -> Gephi офіційний Neo4j plugin

Мета: імпортувати граф з Neo4j, де лежать дані GroupInt, у Gephi Desktop для інтерактивної візуалізації та експорту.

Передумови

Покроково — підключення

ВкладкаПризначення
Labels & Relationship typesСписки міток і типів ребер через CALL db.labels().
Nodes & Edges queriesКастомний Cypher для імпорту / тесту.
Тестовий запит MATCH (n) RETURN id(n), labels(n) не заповнює список міток на першій вкладці.

Покроково — імпорт, режим A

Альтернатива — режим B, Custom Cypher

Граф «учасник ↔ група» без усіх повідомлень.

Nodes:
MATCH (u:User) RETURN id(u) AS id, labels(u) AS labels

Edges:
MATCH (u:User)-[r:MEMBER_OF]->(g:Group)
RETURN id(r) AS id, type(r) AS type, id(u) AS sourceId, id(g) AS targetId

Перевірка

docker exec groupint-neo4j cypher-shell -a bolt://localhost:7687 \
  "CALL db.labels() YIELD label RETURN label ORDER BY label"

Типові помилки

ПроблемаЩо робити
Verify failedПеревірити порт 17687, не 7687; контейнер groupint-neo4j running.
Порожні списки labelsСпочатку Verify; не покладатися на вкладку Nodes & Edges для discovery.
Gephi зависаєЗанадто великий граф — імпортувати без Message, додати LIMIT у Cypher.
Auth failedУ desktop stack NEO4J_AUTH=none — не вводити neo4j/password.

Розділ 5 — gephi-ai + Claude

Мета: керувати вже імпортованим графом у Gephi через AI-асистента Claude: статистика, спільноти, layout, експорт PNG/PDF.
Ланцюг: GroupInt -> Neo4j -> Gephi (Neo4j plugin) -> gephi-ai -> Claude. gephi-ai не підключається до Neo4j напряму.

Передумови

DoneКомпонентВимога
Gephi Desktop0.10.1+
JDK11+ (Adoptium)
Mavenзбірка Java-плагіна
Python3.10+ для MCP server
ClaudeClaude Code або Claude Desktop
API ключAnthropic, окремо від Telegram

Архітектура

Claude (Code / Desktop)
        |
   MCP Protocol (stdio)
        |
   gephi-mcp (Python)          <- MCP server
        |
   HTTP http://127.0.0.1:8080
        |
   gephi-mcp-plugin (Java)     <- всередині Gephi Desktop
        |
   Gephi Desktop               <- має бути запущений і з графом

cd gephi-ai/gephi-mcp-plugin

Типові помилки

ПроблемаЩо робити
Health check failedGephi не запущений або plugin не встановлений / не перезапущений.

Глосарій

ТермінПояснення
OTPОдноразовий код від Telegram для входу / GroupInt.
api_id / api_hashКлючі застосунку з my.telegram.org для Telethon.
StringSessionЗбережений токен Telethon у файлі .string, без повторного OTP.
canonical Group.idСтабільний id групи в Neo4j: username або peer:{id}.
BoltПротокол Neo4j для Gephi та драйверів: bolt://host:port.
MCPModel Context Protocol — міст між Claude і gephi-mcp.
MEMBER_OFЗв'язок User -> Group, учасник групи.
ENDORSESЗв'язок Group -> Group, посилання з повідомлень.
