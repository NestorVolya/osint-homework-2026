## Крок 1 — Збір контексту (Jina Reader)

**Інструмент:** Jina Reader API (`r.jina.ai`) — безкоштовний fetch  
**Вхід:** URL підозрілого джерела з Wikipedia-приміток Чекаля  
**Запит:**

```bash
curl -s "https://r.jina.ai/https://pravmir.ru/kalligrafiya-v-tserkvi/" \
  -H "Authorization: Bearer $JINA_API_KEY" \
  -H "Accept: text/plain" > pravmir.ru_chekal_2013-01-22.txt
```

**Вихід:** 150 рядків чистого тексту, збережено локально  
**Час:** ~3 секунди  
**Вартість:** $0

*📸 Скріншот: термінал з командою + розмір файлу після завантаження*

## Крок 4 — Синтез і gap-аналіз (Claude)

**Інструмент:** Claude  
**Вхід:** таблиця з кроку 3  
**Промпт:**

```
На основі верифікованих даних нижче:
[таблиця з кроку 3]

1. Напиши аналітичний профіль (5–8 речень), уникай тверджень confidence < 7
2. Перерахуй gaps — що залишилось невстановленим
3. Для кожного gap — де і як шукати далі
```

**Вихід:**

**Аналітичний профіль:**  
Чекаль Олексій Георгійович — харківський каліграф і дизайнер, арт-директор PanicDesign, задокументована системна співпраця з інституціями РПЦ/МП у період 2008–2022. Зафіксовано участь у щонайменше 4 різних РПЦ-пов'язаних організаціях (ПСТГУ, Артос, Валаамський монастир, Russia Cristiana) через виставкові, дизайнерські та каліграфічні проєкти. "Міжнародні" проєкти в Італії (Rimini Meeting) фінансувались і організовувались структурами московської православної мережі. Після 24.02.2022 — GAP:SILENCE (відсутні публічні заяви про розрив з партнерами).

**Gaps:**
- Чи є матеріали Чекаля на РПЦ-ресурсах ПІСЛЯ 24.02.2022?
- Як оплачувались проєкти Russia Cristiana?

## Порівняння: один великий запит vs pipeline

| Параметр | Один мегазапит | 4-крокований pipeline |
|---|---|---|
| Токени Claude | ~280 000 (читає всі тексти) | ~4 000 (читає лише summary) |
| Вартість | ~$0.84 | ~$0.012 |
| Аудитована логіка | ❌ чорна скринька | ✅ кожен крок перевіряється |
| Точність | ⚠️ контекст змішується | ✅ крок 3 верифікує крок 2 |
| Gaps | ❌ зазвичай не формулює | ✅ explicit список |

**Ключовий висновок:** pipeline не просто дешевший — він **аудитований**.  
Можна перевірити кожен крок і побачити де з'явилась помилка.
